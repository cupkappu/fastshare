# 共享文本框功能测试报告

## 测试概述

本次测试验证了 FastShare 应用的共享文本框功能，支持两种模式：

- **Relay 模式（普通模式）**：通过 WebSocket 服务器中转文本消息
- **P2P 模式**：通过 WebRTC data channel 直接传输文本
